# ShopBag
ShopBag is a Biding Marketplace project Created with MERN Stack Technology

## Backend technology
* Node.js
* Express.js

## Frontend Technology 
* React.js

## Database 
* MongoDB

## Styles
* Tailwindcss
* Ant Design


## Available Scripts

In the project directory, you can run:

### `npm start`

you can install this project:

### `npm install`
